# SayF Pool Control

Run locally from this folder:

```bash
python run_pool.py
```

Windows double-click option:

```text
start_windows.bat
```

Mac/Linux terminal option:

```bash
bash start_local.sh
```

Direct original app path:

```bash
python pool_app_verified/run_pool.py
```

Default passwords:

- AM: `am` or `AM`
- PM: `pm` or `PM`
- Admin: `admin`
- Custom timer password: `pool`
